# n8n с Puppeteer

Docker-контейнер для n8n с поддержкой Puppeteer и Browserless для веб-автоматизации.

## Что это

- **n8n** - платформа для автоматизации рабочих процессов с открытым исходным кодом
- **Puppeteer** - управление браузером Chrome для веб-скрейпинга и автоматизации
- **Browserless** - удаленный Chrome браузер для Puppeteer операций
- **Node.js 22** - последняя версия JavaScript runtime
- **Debian Bookworm** - лёгкий базовый образ Linux

## Архитектура

Проект состоит из двух сервисов:
- **n8n** - основной сервис автоматизации
- **browserless** - сервис для удаленного управления Chrome браузером

## Быстрый запуск

```bash
docker-compose up --build
```

n8n будет доступен по адресу: http://localhost:5678
